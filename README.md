### Getting Started

1. First of all you need to clone Coronatime repository from github:

```sh
git clone https://github.com/RedberryInternship/tazokenkebashvili-coronatime-front.git
```

2. Create .env file

```sh
cp .env.example .env
```

3. Next step requires install all the dependencies.

```sh
npm install
```

or 

```sh
yarn
```
